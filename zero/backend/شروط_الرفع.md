# شرح رفع الباك اند على Railway

## أولا: إرفع الكود على GitHub
```
git add backend/
git commit -m "Add backend"
git push
```

## ثانيا: Connect Railway بـ GitHub
1. افتح https://railway.app/dashboard
2. اضغط **New Project** → **Deploy from GitHub repo**
3. اختار الريبو بتاعك (اللي فيه backend/)
4. في **Root Directory** حط: `backend`
5. اضغط **Deploy**

## ثالثا: حط المتغيرات (Environment Variables)
في صفحة المشروع على Railway:
1. من القائمة اليمين، اضغط على **Shared Variables**
2. هتظهر لك صفحة فيها خانتين:
   - **Variable Name** (الاسم)
   - **Variable Value** (القيمة)
3. كل مرة اضغط **Add Variable** وحط اسم وقيمة
4. كرر 3 مرات عشان تحط:

## رابعا: استنى شوية
Railway هيعمل Build و Deploy تلقائي. خذ راحة لمدة دقيقتين.

## خامسا: جرب
بعد ما يخلص، افتح الرابط اللي Railway يديّك (حاجة زي `https://backend.up.railway.app`)

جرب الرابط ده في المتصفح:
```
https://backend.up.railway.app/api/v1/health
```

لو ظهر لك `{"status":"ok"}` → مبروك، الباك اند شغال 🎉

## سادسا: عدل على Flutter
في ملف `lib/services/api_service.dart`، غير الرابط القديم للرابط الجديد:
```dart
String _baseUrl = 'https://backend.up.railway.app/api/v1';
```
